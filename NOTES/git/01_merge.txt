Для слияния веток в Git нужно выполнить следующие шаги:

1. Переключитесь на ту ветку, в которую вы хотите слить изменения другой ветки.
      git checkout <название_ветки_в_которую_сливаем>
   

2. Слейте другую ветку с текущей выбранной веткой.
      git merge <название_ветки_которую_сливаем>
   

3. Разрешите конфликты, если они возникнут. Git автоматически попытается слить изменения, но если возникнут конфликты, вам придется решить их вручную.

4. После разрешения конфликтов и завершения слияния, необходимо закоммитить изменения.
      git commit -m "Merge branch <название_ветки_которую_сливаем>"
   

5. Отправьте изменения на удаленный репозиторий, если это необходимо.
      git push
   

Теперь ветки успешно слиты вместе.
